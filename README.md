## Getting Started: Backend  
1. move to the root directory  
```bash
$ cd cc26-senior-project-backend
```
2. download dependencies  
```bash
$ yarn install
```
3. run the server
```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
